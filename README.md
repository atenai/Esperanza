# Esperanza

「エスペランサ」

（概要・テーマ）
この作品は、UnrealEngine5を勉強するために開発している作品になります。
現在も開発中のTPSゲームで最終的にはサバイバルな旅をしながらゴール目指すTPSゲームにする予定です。

(開発時期・期間)
DICO株式会社時の2023年9月～現在も開発中

（開発ツール）
・ゲームエンジン：UnrealEngine5(5.1.1)
・言語：ブループリント

（チームメンバー）
プログラマー1人（自分）

（ゲームシステム）
敵を倒しながらゴールを目指すTPSゲームです。
ゴールに着くとゲームクリアーになります。
体力が0になるとゲームオーバーになります。

(操作方法)
・W・S・A・Dキーでプレイヤーキャラクターを移動させることができます。
・Spaceキーでプレイヤーキャラクターをジャンプさせることができます。
・Rキーでリロードができます。
・マウスの左ボタンで射撃ができます。
・マウスの右ボタンでエイムができます。
・Pキーでポーズ画面を開けます。

(起動方法)
「Esperanza(ビルド)」というフォルダーを開いてください。
その中にある「Esperanza」というアプリケーションをダブルクリックしてください。
